Cara mendownload dan menginstall software Rstudio (R editor)

2016/07/03

Setelah R terinstal dengan benar di komputer, R dapat digunakan langsung. Kode-kode perintah dapat ditulis langsung di jendela console. Kelemahnya, setelah mengeksekusi kode perintah ini, pengubahan (editing) sangat tidak nyaman karena harus menggunakan tanda panah keatas baru diedit. Hal ini dapat diatasi dengan menggunakan jendela R editor.

Jendela editor dapat di buka dengan membuka file > new script, kemudian akan muncul jendela R editor. Kode perintah dapat ditulis disini, penekanan tombol enter akan memberikan efek pergantian baris (enter di R console berarti eksekusi). Semua perintah di R editor ini tidak akan dieksekusi oleh R, kecuali kode perintah ini dipindah ke R console. Cara memidahkannya dapat dengan menggunakan cara klik kanan dan pilih run line or selection atau dengan cara copy dan paste.

R editor juga ditawarkan oleh beberapa pihak ketiga seperti Rstudio dan Tinn-R yang juga gratis. Kelebihan R editor adalah lebih mudah dan menarik karena kode perintahnya dapat auto fill dan berwarna. Saya lebih memilih Rstudio dari pada Tinn-R karena tampilan Rstudio yang menarik dan mudah apalagi jika ingin mengintegrasikan dengan Latex.

Berikut cara mendownload Rstudio untuk windows:

  1. Buka alamat ini
  2. Klik pada Rstudio untuk windows

Cara menginstall:

  1. Klik kanan pada ikon Rstudio dan pilih run as administrator.
  2. Kemudian pilih next > next > install > finish

Jika sudah diinstal dengan benar Rstudio dapat dijalankan. Saat pertama kali menjalankan jendela terlihat seperti (1). Kemudian untuk memulai file baru, pilih file > new script (2). Jendela Rstudio yang baru akan muncul (3).

Didalam Rstudio ada 4 jendela yaitu, jendela editor (3a) untuk menuliskan kode perintah, jendela console (3b) tempat kode perintah dieksekusi, jendela environment (3c) tempat daftar objek-objek yang sedang aktif, dan jendela files, plots, packages, help, viewer (3d) tempat melihat root file, hasil plot, paket yang diinstal, dan help (bantuan). Setiap kode perintah di jendela editor dapat dieksekusi dengan menekan alt + enter atau menggunakan tombol run dikanan atas (4).
