Status of Food Security in Banten Province, Indonesia

Weksi Budiaji

2017/10/01

Budiaji, W., Sari, R. M., Suherna. 2017. Status of Food Security in Banten Province, Indonesia. In: Proceedings 1st International Conference on Food Security Innovation. Project Implementation Unit - Islamic Development Bank: University of Sultan Ageng Tirtayasa, pp.Â 11-19. (Download)

Alhamdulillah telah terbit makalah saya di Proceedings of International Conference on Food Security Innovation. Makalah ini bercerita tentang kondisi ketahanan pangan di Provinsi Banten .

Saya menggunakan penelitian deskriptif kuantitatif untuk mengeksplorasi kondisi ketahanan pangan di Provinsi Banten. Karya saya ini bermanfaat untuk mengetahui tingkat ketahanan pangan tiap kabupaten sehingga program spesifik dapat dititik beratkan pada lokasi-lokasi tertentu. Penelitian ini dapat dikembangkan untuk wilayah lain dan penambahan variable yang lebih beragam. Saya mengundang pembaca untuk bekerjasama dalam pengembangan penelitian ini.

Terima kasih saya ucapkan kepada Kementrian Ristek dan DIKTI dan OeAD atas dukungan penuhnya kepada riset saya melalui program Indonesia Austria Scholarship Program (IASP).

Selamat membaca. Mohon masukannya.
