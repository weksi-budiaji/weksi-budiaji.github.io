Skala Pengukuran dan Jumlah Respon Skala Likert

Weksi Budiaji

2013/12/01

Budiaji, W. 2013. Skala Pengukuran dan Jumlah Respon Skala Likert. Jurnal Ilmu Pertanian dan Perikanan Vol.2 (2): 125 - 131. (Download)

Alhamdulillah telah terbit makalah saya di Jurnal Ilmu Pertanian dan Perikanan. Makalah ini bercerita tentang skala likert.

Saya menggunakan studi literatur untuk menjelaskan penggolongan skala likert. Karya saya ini bermanfaat untuk mengidentifikasi bahwa skala likert dapat digolongkan menjadi skala interval. Jumlah respon yang disarankan untuk skala likert adalah tujuh. Penelitian ini dapat dikembangkan untuk aplikasi penggunaan skala likert. Saya mengundang pembaca untuk bekerjasama dalam pengembangan penelitian ini.

Selamat membaca. Mohon masukannya.
