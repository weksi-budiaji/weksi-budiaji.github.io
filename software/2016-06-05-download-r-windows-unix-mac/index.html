Cara mendownload dan menginstal software R (Windows, Mac, dan Linux/ unix)

2016/06/05

Saat ini, keseharian kita tidak dapat terlepas dari teknologi. Teknologi ini terus berkembang dengan tujuan untuk mempermudah pekerjaan manusia. Komputer salah satunya yang memberikan banyak kemudahan untuk kita termasuk dalam hal analisis data.

Data yang sudah ada ditangan biasanya dianalisis terlebih dahulu sebelum ditampilkan dan sering kali kesimpulan-kesimpulan juga ditampilkan berdasarkan data, misalkan ditampilkan deskriptif statistik dan uji t. Untuk mempermudah analisis, software statistik dapat digunakan. Banyak software statistik yang ada, dari yang gratis hingga yang berbayar lisensi mahal tersedia. Yang akan digunakan kali ini adalah software statistik gratis yaitu software R.

Apa itu software R? Sistem R untuk statistika komputasi adalah salah satu sistem yang paling populer untuk analisis data dan visualisasi grafik yang sangat bervariasi. R adalah sistem yang paling fleksibel untuk data manajemen dan analisis. Karena setiap obyek di R mempunyai atribut, pemahaman tentang atribut sangat penting saat data manajemen dan analisis.

Sistem R pada awalnya dikembangkan oleh R. Ihaka dan R. Gentleman. Bahasa R sama dengan bahasa S yang dikembangkan oleh AT and T Bell Laboratories. Tetapi lisensi S secara eklusif dimiliki oleh Insightful Corporation yang distribusinya dikenal dengan S-PLUS. Kebanyakan perintah di R dapat dijalankan di S-PLUS tanpa perlu pengubahan.

Keuntungan menggunakan R diantaranya adalah gratis sehingga tidak perlu membayar. R dapat diinstall di banyak platform seperti windows, mac, dan linux (unix). R adalah bahasa pemograman yang powerful dan mudah. Selain itu, sebagai open source software dapat dikembangkan. Kemampuan grafis di R juga dapat dibilang handal serta R didukung komunitas yang besar. Jika ada pertanyaan-pertanyaan kita dapat bertanya dikomunitas ini.

Sistem R umumnya menggunakan baris-baris perintah (command) dengan menuliskan text kemudian meminta R untuk menjalankannya. Software statistik yang menggunakan menu (dipilih dengan mouse) mungkin nyaman untuk perintah yang terbatas, tetapi R dengan baris-baris perintah kemampuannya menjadi tidak terbatas. Kita dapat melakukan banyak hal dengan baris-baris perintah tersebut.

Kenapa harus menginstall dan menggunakan R? Karena R menyediakan kebebasan mengontrol dan mendesain analisis dari yang sederhana hingga yang komplek. Kekuatan ini tidak banyak diberikan pada software statistik yang lain.

Pertama kali software R diinstall, hanya paket base yang termasuk didalamnya. Untuk menganalisis statistik yang umum seperti ukuran pemusatan (mean, median), penyebaran atau uji-uji yang umum seperti uji t, chi-squared, anova, paket ini lebih dari cukup. Tetapi untuk analisis yang lebih kompleks seperti analisis geostatistik, functional data, dll perlu tambahan paket lain yang dapat dilakukan setelah R terinstal dengan benar.

Berikut cara mendownload dan menginstall R untuk windows:

  1. Buka http://www.r-project.org/
  2. Klik mirror, untuk memilih berdasarkan negara
  3. Pilih yang dekat dengan posisi sekarang
  4. Klik Windows dibawah Download and Install R
  5. Klik pada base
  6. Klik pada link untuk download pada versi terakhir R (.exe file)

  1. Saat download selesai, cek file dan double klik pada ikon .exe
  2. Pilih bahasa inggris klik OK. next > next > next > (pilih yang 32 atau 62 bit) kemudian next > next > next > next > terakhir finish

Jika R terinstal dengan benar, user tinggal mengklik pada gambar ikon R untuk menjalankan software R. Jendela yang muncul jika ikon R di klik dapat seperti ini

Kita dapat mengetikan 1 + 1 kemudian enter, atau 2 * 3 kemudian enter atau 1 + 2 *3 kemudian enter. Lihat dan evaluasi hasilnya.

Installasi untuk Mac OS mirip dengan windows. Cara mendownload dan menginstall R untuk platform Mac OS tidak jauh berbeda, hanya pada langkah ke-4 diatas pilih Download R for Mac OS kemudian file yang didownload adalah .pkg

Berbeda dengan linux/ unix yang harus menyeting dulu source list. Berikut cara donwload dan install R di linux:

  1. Kebanyakan linux distribusi (seperti ubuntu/ mint, atau red hat), instalasi R dapat menggunakan terminal dengan kendali admin (sudo). Setting dulu source list untuk download paket-paket nantinya (ubuntu trusty)
$sudo sh -c 'echo "deb http://cran.rstudio.com/bin/linux/ubuntu trusty/" >> /etc/apt/sources.list'

Jika versi linux lain yang dipakai silahkan mengecek di sini

  1. Masukkan public key
$gpg -keyserver keyserver.ubuntu.com -recv-key E084DAB9
$gpg -a -export E084DAB9 | sudo apt-key add -
  1. Baru instal R
$sudo apt-get update
$sudo apt-get -y install r-base

Untuk menjalankan R dapat menggunakan terminal ketik R, atau mengklik di bagian menu.
