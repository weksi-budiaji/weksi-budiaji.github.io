Medoid-based Shadow Value Validation and Visualization

Weksi Budiaji

2019/07/01

Budiaji, W. 2019. Medoid-based Shadow Value Validation and Visualization. International Journal of Advances in Intelligent Informatics Vol. 5 (2), pp.Â 76-88. (Download)

Alhamdulillah telah terbit makalah saya di International Journal of Advances in Intelligent Informatics. Makalah ini bercerita tentang metode validasi dan visualisasi hasil analisis cluster.

Saya menggunakan metode simulasi untuk membuat metode validasi dan visualisasi hasil cluster berdasarkan pada medoid (cluster center). Karya saya ini bermanfaat untuk memvalidasi dan memvisualisasikan hasil analisis cluster. Penelitian ini dapat dikembangkan untuk aplikasi data riil. Saya mengundang pembaca untuk bekerjasama dalam pengembangan penelitian ini.

Terima kasih saya ucapkan kepada Kementrian Ristek dan DIKTI dan OeAD atas dukungan penuhnya kepada riset saya melalui program Indonesia Austria Scholarship Program (IASP).
